### 1.4.1 선언적 API

- 선언적(Declarative) API
- 컨테이너가 어떤 상태가 되어야 하는지 선언하면 쿠버네티스가 그 상태를 유지하도록 함

<br>

- 앱
  - 단일 배포로 구성한 파드(pod)
  - 컨테이너
- 서비스
  - 여러 개 앱을 합한 것

<br>

- 단점
  - 앱 재시작 같은 단순한 작업은 k8s에서 할 수 없음
    - 앱 재시작은 시스템 내부 동작
  - 하지만 kubectl 1.5 부터 디플로이먼트, 스테이트풀세트, 데몬세트에 재시작 기능 추가됨
